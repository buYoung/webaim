기본설정

고정방식
x축 위치
-20~20 사이의 값을 입력해주세요
y축 위치
-20~20 사이의 값을 입력해주세요

위의 기능은 감지된 색으로부터 + 혹은 - 의 위치로 추가적으로 이동합니다.(보통 창모드일경우 사용)

위의 기능은 감지된 색이 마우스와 지정한 값보다 거리가 멀경우 지정한값으로 바꿉니다.

랜덤작동
0~100 사이의 값을 입력해주세요

위의 기능은 색상을 찾은후 작동할때 확률적으로 작동합니다.

딜레이
0~10000 사이의 값을 입력해주세요
자동클릭 발사횟수
0~10 사이의 값을 입력해주세요
자동클릭 딜레이
0~100 사이의 값을 입력해주세요

위의 기능은 지정된 값만큼 색상찾은후 멈춥니다 (1초 = 1000 // 0.001초 = 1)

자동클릭만 사용하기
몸전용
인식색상의 몸이 적힌 색상만 인식합니다.
머리전용
인식색상의 머리라고 적힌 색상만 인식합니다.
기타색만적용
인식색상의 기타색이 적힌 색상만 인식합니다.
w키 누를때 a / d 키누르기
색상찾는 갯수
0~50 사이의 값을 입력해주세요
한 화면안에 찾는 색상의 갯수를 정합니다. 높을수록 느려짐 낮을수록 정확하지않음.
